 Задача дипломного проекта Автоматизировать UI- и API-тесты из вашей финальной работы по ручному тестированию.

**Шаги работы**
Подключите зависимости:
selenium,
requests,
pytest,
allure,
остальные зависимости на ваше усмотрение.
В проекте создайте файл 
test\test_ui.py
 с UI-тестами.
В проекте создайте файл 
test\test_api.py
 с API-тестами.
В коде не должно быть специфичных данных — вынесите их в файл:
настройки окружения (URL, пути до файлов и т. д.);
тестовые данные (логины, пароли, токены и т. д.).
